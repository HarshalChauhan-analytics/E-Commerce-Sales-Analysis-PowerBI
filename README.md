# Madhav E-Commerce Sales Dashboard 🛒📊

## Project Overview
This project features an interactive Power BI dashboard designed to analyze the sales performance and customer purchasing behavior for an e-commerce platform. The dashboard provides stakeholders with a clear, high-level overview of critical business metrics, identifying the most profitable product categories, seasonal trends, and preferred payment methods.

## Core Business Insights & Visualizations
* **Monthly Profitability:** A column chart tracking profit trends across the calendar year to identify peak purchasing seasons.
* **Category Performance:** Donut charts breaking down order volume by overarching categories (Clothing: 62%, Electronics: 20%, Furniture: 16%).
* **Sub-Category Deep Dive:** Bar charts highlighting the top 5 most profitable specific sub-categories (Printers, Bookcases, Sarees, Accessories, Tables) alongside geographical performance by State.
* **Customer Payment Preferences:** Analysis of transaction types showing a strong preference for Cash on Delivery (COD at 43.7%) followed by UPI and Debit cards.

## Technical Skills Demonstrated
* **Data Import & Cleaning:** Extracted and structured flat-file e-commerce data for accurate visual reporting.
* **Dashboard UI/UX Design:** Built a highly readable, contrasting layout using a deep purple aesthetic with bright neon accents for optimal data scannability.
* **Interactive Filtering:** Included a dynamic slicer allowing users to filter the entire report by specific States to track regional performance.
